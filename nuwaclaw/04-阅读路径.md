## 关键设计速查

| 问题 | 答案 |
|------|------|
| 多项目并发 AI 对话怎么不互相干扰？ | `engines: Map<projectId, AcpEngine>`，每个项目独立进程 |
| 引擎进程 env 怎么不泄漏用户 .zshrc 配置？ | `createIsolatedEnvironment`：清洗所有 `CLAUDE_*`/`ANTHROPIC_*`，显式注入 |
| nuwaxcode 的首次 chat 为什么快？ | warmup 池预先 spawn 进程，命中 `tryReuse` 则跳过 ~2s 冷启动 |
| claude-code 和 nuwaxcode 配置怎么传给引擎？ | claude-code：命令行 + 隔离 HOME 的 settings.json；nuwaxcode：`OPENCODE_CONFIG_CONTENT` 环境变量（JSON）|
| Agent 写文件为什么不弹权限对话框？ | `handlePermissionRequest` 自动 `allow_always`/`allow_once`；只有 `question` 类型被 `cancelled` |
| 记忆功能默认关，怎么开？ | `request.open_long_memory = true`，nuwax-backend 下发或用户设置 |
| MCP 配置有两个来源怎么合并？ | SQLite 本地配置（基础）< ACP `context_servers`（request 下发，优先级更高）|
| Windows 上 Agent 写系统目录怎么拦截？ | strict 模式：`tool_call_update` 前置拦截 → `evaluateStrictWritePermission` → 超出 `writableRoots` 立即 abort |
