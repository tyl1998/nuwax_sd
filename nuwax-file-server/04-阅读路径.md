## 关键设计速查

| 问题 | 答案 |
|------|------|
| 多个项目并发启动，端口怎么不冲突？ | `portPool` 单例，Set O(1) 分配 |
| 子进程退出后日志流怎么安全关闭？ | `safeWrite()` 检查 `stream.destroyed`，`child.on('exit')` 延迟 200ms 关流 |
| Alpine（musl）上 Vite 原生模块报错怎么处理？ | 检测 glibc 版本，不匹配则 `removeNodeModules` 清理重装 |
| AI Agent 更新代码时如何避免删掉系统配置文件？ | `pruneMissingFiles` 保护 `CONTENT_TRAVERSE_EXCLUDE_FILES`（AGENT.md/CLAUDE.md）|
| zip 回滚怎么防路径穿越攻击？ | `path.resolve(targetPath).startsWith(resolvedProjectPath + sep)` 校验 |
| 动态添加的技能怎么在重建工作空间时保留？ | `.dynamic_add.lock` 文件标记，`createWorkspace` 先 mv 暂存再恢复 |
| 跨挂载点 rename 失败怎么办？ | `EXDEV` 错误时回退为 `copyRecursive + rm -rf` |
